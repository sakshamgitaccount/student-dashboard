# Student Dashboard

A React application to view, filter, and add students—protected by Firebase Email/Password authentication—with a responsive Tailwind CSS UI.

## Demo Video

https://drive.google.com/file/d/1BW20OfRuc-fj1wykf0AuUtIb253n1MB-/view?usp=sharing

---

## Features

- **List Students** — Fetches and displays mock data via `axios-mock-adapter`.  
- **Filter** — Dropdown to filter by course.  
- **Add Student** — Protected route; only signed-in users can add new students.  
- **Details** — Click a student to view their details on a separate page.  
- **Firebase Auth** — Email/Password sign-in via Firebase.  
- **Responsive Design** — Mobile-first layout using Tailwind CSS.
